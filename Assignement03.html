HTML for Lists

    Start tags for

    • an ordered list

      • < ol >< /ol >

    • an unordered list

      • < ol >< /ol >

    • a description list

      • < dl >< /dl > defines the description list

    • a list item

      • < li >< /li >

    • a description term

      • < dt >< /dt > defines the term (name)

    • a description data

      • < dd >< /dd > describes each term

Css for lists

    • Make a bullet square

      • ul.b { list-style-type: square; }

    • Make bullet an image named "my-bullet.png"

      • ul { list-style-image: url("my-bullet.png"); }

    • Make bullet an uppercase roman number

      • ol.c {list-style-type: upper-roaman; }

    • Set bullet position to be inside the list item's text flow

      • ul.b {list-style-position: inside; }

HTML for tables

    Start tags for

    • a table

      • < table >

    • a table row

      • < tr >

    • a table header

      • < thead >

    • a table header cell

      • < th >

    • a table body

      • < tbody >

    • a table cell

      • < td >

    • a table footer

      • < tfoot >

    • a table footer cell

      • < tfoot >

Css for tables

    • Black dotted border of 1px width

      • table, th, td { border:1px solid (dotted) black; }

    • Solid border of 1px width having a red color and rounded corners

      • table, th ,td { border-radius:5px; border: 1px solid red; }

    • Merge the borders of table cells so that they appear as on line

      • table { border:collapse; } table, th ,td { border: 1px solid black; }
